```
a = len('egoing')
print(a)
```
###### 단어의 전체 길이를 알고 싶을 때 사용하는 것은 len()이다.

---

```
b = len(['a','b','c'])
print(b)
```
###### len()은 string에서 뿐만 아니라 list에서도 사용가능하다.

---

```
c ='egoing'.capitalize()
print(c)
```
###### 단어의 첫글자를 대문자화하는 기능은 .capitalize()이다.

---

```
d = ['jiwon', 'gwangju', 'student']
print(d[0])
person = {'name':'jiwon', 'address':'gwangju'}
print(person['name'])
```
###### python에서는 숫자를 0부터 센다.
