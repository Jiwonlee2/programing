[Comparison operater]


[in]

```
print('world' in 'Hello world') #True
```
###### in은 뒤에 있는 값에 앞에 있는 값이 포함되어 있는지를 알기 위한 기능이다. 
###### a in b가 true가 되기 위해선 b는 a를 포함하고 있어야 한다.
